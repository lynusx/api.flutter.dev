# kMinInteractiveDimension

```dart
double kMinInteractiveDimension
```

根据 Material 设计规范，任何交互区域的最小尺寸。

用于避免出现难以让用户交互的过小区域。它适用于区域的两个维度，因此边长为 kMinInteractiveDimension x kMinInteractiveDimension 的正方形是应响应手势的最小可接受区域。
